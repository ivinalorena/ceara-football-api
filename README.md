
# **CearáTrack**

**CearáTrack** é uma aplicação web desenvolvida em **Python/Flask** que consome a API pública **Football-Data.org** para exibir estatísticas completas e atualizadas sobre o **Ceará Sporting Club**.
O projeto reúne informações como partidas, resultados, status dos jogos e outros dados relevantes, apresentados em uma interface simples.

O objetivo principal do desenvolvimento é aprofundar meus estudos pessoais sobre consumo de APIs e integração com aplicações web.

---

## **Funcionalidades**

* Consulta de dados em tempo real da API Football-Data.org
* Exibição de:

  * Últimos jogos
  * Próximas partidas
  * Resultados completos
  * Informações gerais das competições
* Interface leve e responsiva com HTML + CSS
* Backend em Flask consumindo e processando dados externos

---

## **Tecnologias utilizadas**

* **Python 3**
* **Flask**
* **Football-Data.org API**
* **HTML5**
* **CSS3**

---

## **API utilizada**

Este projeto utiliza a **Football-Data.org**, uma API gratuita para dados de futebol.
Documentação oficial: [https://www.football-data.org/](https://www.football-data.org/)
